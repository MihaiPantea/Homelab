# Windows Infrastructure Homelab

## Overview

This repository documents my Windows Server homelab built in Proxmox.

The goal is to develop the skills required for a Windows Infrastructure / 2nd Line Engineer role.

## Environment

### Hypervisor

- Proxmox VE

### Domain

- homelab.internal

### Servers

- DC01
    - Active Directory
    - DNS
    - DHCP

### Clients

- WINCLIENT
